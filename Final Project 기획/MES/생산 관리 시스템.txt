✅ [4] 생산 관리 시스템 – MiniMES
🎯 프로젝트 목적
소규모 제조업체의 생산/재고/공정 관리를 디지털화

품질 기록, 작업 지시, 실시간 대시보드 제공

👥 주요 사용자
관리자(공장장), 작업자, 품질검사원

🧩 주요 기능
역할	기능
관리자	공정 계획, 자재 등록, 작업 지시 생성
작업자	실시간 작업 현황, 상태 변경
검사자	품질 점검 결과 입력, 불량 등록
시스템	생산 통계 대시보드 자동 생성

🔧 기술 구조
Java17 + Spring Boot + JPA + MySQL

Vue.js + Chart.js

Redis(작업 상태 캐시), MongoDB(로그, 품질 이력)

Docker + Azure VM

선택: 작업 예측 AI(기초 불량률 예측)

🤖 AI/외부 API
불량률 예측 모델(단순 Regression 모델로 시작 가능)

외부 센서 API 연동은 mock 처리 가능

⭐ 차별성
생산+재고+품질 통합

불량률 예측 및 대시보드로 시각적 효과

✅ 기대효과
경량 MES로 중소 제조사 대상 상품화 가능

개발 기간 대비 기능 구성이 균형 잡힘
