프로젝트 기획: AI 기반 맞춤형 학습 관리 및 콘텐츠 추천 시스템 (가칭: EduAI)
프로젝트 개요:

목표: 학생 개개인의 학습 데이터를 AI로 분석하여 맞춤형 학습 경로를 제시하고, 최적의 학습 콘텐츠를 추천하며, 학원 관리의 효율성을 높이는 Java 기반 시스템을 개발합니다. 'Alditor'가 학습 격차 해소에 기여하듯이, 'EduAI'도 같은 목표를 가집니다.
핵심 가치: 개인화된 학습 경험 제공을 통한 학업 성취도 향상 및 학원 운영의 효율성 증대.
대상: 중소규모 학원 및 교육 기관, 자기주도 학습을 원하는 학생.
1.5개월 목표 MVP 핵심 기능:

학생/강사/관리자 회원 관리: 기본 정보 등록/수정, 로그인 (소셜 로그인 제외).
강의/문제 관리: 강좌 정보 등록, 문제 은행 구축 (객관식/주관식), 문제 난이도 설정.
학습 진도 관리: 학생별 강의 수강 진도율 기록, 문제 풀이 기록 저장.
간단한 AI 기반 취약점 분석 및 콘텐츠 추천: 학생의 문제 오답률을 기반으로 취약한 단원 식별, 해당 단원의 다른 문제나 강의 콘텐츠 추천 (기본적인 연관성 기반). Python 연동을 통한 AI 로직.
강사용 학생 학습 현황 대시보드: 소속 학생들의 전반적인 진도율 및 성적 추이 확인.
기술 스택 활용 방안:

Backend: Java 17, Spring Boot, JPA.
Database:
MySQL/PostgreSQL/OracleDB: 학생, 강사, 강좌, 수강 정보, 문제 정보 등 관계형 데이터.
MongoDB: 학생별 학습 로그 (어떤 문제에서 오랫동안 머물렀는지 등), AI 분석 결과 원본 데이터.
Caching: Redis: 실시간 접속자 수, 인기 강의 목록 등 캐싱.
Frontend: React/Vue.js (학생용 학습 대시보드, 강사용 관리 페이지).
AI 연동: Python (취약점 분석, 콘텐츠 추천 알고리즘) 및 Spring Boot와의 REST API 통신.
Containerization: Docker/Podman (각 서비스 컨테이너화).
Cloud: Azure (VMs, Azure SQL Database/PostgreSQL, Azure Cosmos DB for MongoDB, Azure Cache for Redis).
향후 확장 고려 사항:

고도화된 AI 기반 학습 경로 추천 및 예상 성적 예측.
개인별 오답노트 자동 생성 및 유사 문제 추천.
AI 튜터 챗봇 (학습 질문 응대).
화상 강의 연동 및 자동 녹화/텍스트 변환.
