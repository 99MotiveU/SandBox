✅ [1] 샵인샵 기반 커머스 플랫폼 – ShopNest
🎯 프로젝트 목적
지역 소상공인들이 개별 쇼핑몰을 만들기 어려운 문제 해결

통합 플랫폼 안에서 각 점포가 자체 페이지를 운영하는 “샵인샵” 커머스 구조

👥 주요 사용자
점주(입점 판매자), 고객(구매자), 관리자

🧩 주요 기능
역할	기능
고객	회원가입/로그인, 상품 검색 및 구매, 장바구니, 주문, 리뷰
점주	샵 등록/관리, 상품 등록 및 재고 관리, 주문 확인, 정산 관리
관리자	전체 샵/상품 모니터링, 신고 처리, 데이터 통계 확인

🔧 기술 구조
Java17 + Spring Boot + JPA + MySQL

React.js (또는 Vue.js) + Axios + TailwindCSS

Redis(세션/인기 상품 캐시), MongoDB(리뷰, 로그)

Python + Flask (상품 추천 API)

외부 연동: 카카오/네이버 로그인, 카카오페이 or 토스페이 API

배포: Docker(Podman) + Azure App Service

🤖 AI/외부 API
상품 카테고리 기반 추천 시스템

카카오페이 연동, 소셜 로그인 API

⭐ 차별성
다중 점포 구조(독립점포 운영 가능)

고객 맞춤형 상품 추천 + 정산 구조까지 구현

✅ 기대효과
실제 커머스 서비스 수준의 구조 → 시연 시 실감도 높음

지역 상권 대상 SaaS 형태 확장 가능
