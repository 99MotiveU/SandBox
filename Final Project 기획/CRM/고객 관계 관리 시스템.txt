✅ [3] 고객 관계 관리 시스템 – Relatia
🎯 프로젝트 목적
고객 응대 자동화 + 고객 이탈 예측

중소기업의 고객 유지율 향상 지원

👥 주요 사용자
상담사, 관리자, 마케팅 담당자

🧩 주요 기능
역할	기능
상담사	고객 등록, 상담 기록, 태그 및 이슈 분류
관리자	고객 등급, 상담 통계, 마케팅 리포트 확인
시스템	자동 분류된 고객 응대 템플릿 추천, 이탈 예측 분석

🔧 기술 구조
Java17 + Spring Boot + JPA + OracleDB

Vue.js + Chart.js

Redis(고객 등급 캐시), MongoDB(이벤트 로그)

Python + NLP 기반 이메일 자동 분류

외부 연동: 이메일 전송 API, 문자 알림 API

배포: Docker + Azure App Service

🤖 AI/외부 API
NLP 기반 고객 문의 자동 분류 (텍스트 분류 모델)

고객 이탈 예측 분석

알림/이메일 자동 발송

⭐ 차별성
AI로 응대 업무 절감 + B2B 활용도 높음

상담 히스토리 기반 분류/등급화 자동화

✅ 기대효과
