'방치된 PM' 해결을 위한 시민 참여형 도시 문제 개선 플랫폼

1. 사회 문제 정의
공유 전동킥보드, 전기자전거 등 개인형 이동장치(Personal Mobility, PM) 이용이 폭발적으로 증가했지만, 관련 제도는 미비하여 사회적 갈등이 심각합니다. 특히 아무렇게나 '방치된' PM은 보행자(특히 교통약자)의 안전을 위협하고 도시 미관을 해치는 주범으로 꼽힙니다. 이용자, PM 운영사, 지자체 모두 책임만 떠넘길 뿐, 실질적인 해결책은 부재한 상황입니다.

2. 기획 의도 및 해결 방안
시민들의 집단지성을 활용하여 방치된 PM 문제를 실시간으로 해결하고, 데이터를 축적하여 더 나은 PM 정책 수립의 근거를 마련하는 시민 참여형 Civic-Tech(시민 기술) 플랫폼을 구축합니다.

3. 핵심 기능
원터치 신고 및 자동 분류:
사용자가 방치된 PM의 사진을 찍어 올리면, **이미지 인식 AI(외부 API 활용)**가 자동으로 해당 PM의 운영사(e.g., 씽씽, 지쿠터)를 식별하고 분류.
사진의 GPS 데이터를 기반으로 지도에 위치를 자동으로 표시.
실시간 공유맵 및 '핫스팟' 대시보드:
신고된 모든 PM의 위치가 실시간으로 지도 위에 시각화.
특정 지역에 신고가 집중되면 '불법 주차 핫스팟'으로 지정하고, 해당 지역의 PM 운영사와 관할 구청에 주기적으로 통계 데이터를 이메일로 자동 리포팅.
'PM 수거 미션' 및 리워드:
(게임화 요소) PM 운영사와 협력하여, 신고된 PM을 지정된 주차 구역으로 옮기는 사용자(일반 시민 또는 전문 수거팀)에게 소정의 포인트나 쿠폰을 제공하는 '수거 퀘스트' 기능.
안전 PM 경로 추천:
축적된 사고 다발 지역, 불법 주차 핫스팟, 도로 공사 정보 등을 피해 가는 'PM 전용 안전 길찾기' 기능 제공.
4. 기술 스택 및 포인트
Backend: Spring Boot, Spring Security
Database: PostgreSQL + PostGIS (공간 데이터 처리), Redis (실시간 위치 정보 캐싱)
AI/API: Google Vision API 또는 Naver Clova OCR/Image Recognition API 연동.
Key Points:
공간 데이터 처리: 사용자의 위치 정보, PM의 좌표 등을 DB에 저장하고, 특정 반경 내의 데이터를 효율적으로 조회하는 쿼리(e.g., ST_DWithin) 작성 경험.
외부 AI 서비스 연동: 이미지 인식 AI를 활용하여 비즈니스 로직을 자동화하는 실전 경험.
게임화(Gamification): 미션, 랭킹, 리워드 시스템을 설계하여 사용자 참여를 유도하는 서비스 기획 및 구현 능력.
5. 기대 효과 및 포트폴리오 가치
'Civic-Tech'라는 새로운 분야의 프로젝트 경험을 통해 사회 문제 해결에 대한 적극적인 의지를 보여줄 수 있음.
지도 서비스, 위치 기반 기술, 이미지 인식 AI 등 최신 기술 트렌드를 프로젝트에 녹여내어 기술적 차별성을 확보.
