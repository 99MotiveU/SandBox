긱워커(Gig Worker) 익명 소득 공유 및 정보 플랫폼

한 줄 요약: 배달 라이더 등 긱워커들이 소득을 익명으로 공유/비교하고, 꿀팁을 나누는 커뮤니티.
기획 배경: 새로운 노동 형태인 긱워커들의 '정보 비대칭'과 '권익 문제'를 해결.
PoC 목표: 이미지(소득 인증샷)에서 OCR로 데이터를 추출하여 DB에 저장하고, 이를 통계로 보여주는 핵심 파이프라인 완성.
핵심 기능:
소득 정산 화면 캡처 이미지 업로드 및 OCR을 통한 데이터 자동 입력
플랫폼별/지역별 평균 소득 통계 대시보드
인증된 사용자만 참여 가능한 익명 게시판
마이크로서비스 분리 방안:
서비스 A (사용자/인증): 긱워커 회원가입, 로그인, 익명성 보장 로직 처리.
서비스 B (데이터/OCR): 이미지 업로드 처리, 외부 OCR API 연동, 텍스트 데이터 파싱 및 저장.
서비스 C (통계/커뮤니티): 저장된 데이터를 분석하여 통계 생성, 익명 커뮤니티 기능 제공.
기대 학습 효과: 외부 AI(OCR) API 연동, 비정형 데이터(이미지)를 정형 데이터로 가공하는 파이프라인 구축.
