'방치된 킥보드' 신고 및 안전경로 제공 플랫폼

한 줄 요약: 시민들이 방치된 공유 킥보드를 신고하고, 데이터를 모아 안전한 PM(개인형 이동장치) 경로를 제공하는 Civic-Tech 플랫폼.
기획 배경: 심각한 도시 문제인 공유 킥보드 문제를 시민 참여(집단지성)로 해결.
PoC 목표: 위치 기반(GPS) 데이터 신고 기능을 구현하고, 신고 데이터를 지도 위에 시각화하는 기능 완성.
핵심 기능:
사진과 GPS 위치 기반의 방치 킥보드 간편 신고
신고된 킥보드 위치 실시간 지도 시각화
신고 핫스팟(Hotspot) 통계 제공
마이크로서비스 분리 방안:
서비스 A (사용자/인증): 일반 사용자, PM 업체 관리자 등 역할 기반 회원 관리.
서비스 B (신고/지도): 신고 데이터(이미지, 좌표) 처리, 지도 시각화를 위한 API 제공.
서비스 C (통계/알림): 지역별/업체별 통계 데이터 생성, 핫스팟 발생 시 관리자에게 알림.
기대 학습 효과: 위치 기반 서비스(LBS) 데이터 처리, 지도 API 연동, 공익적 Civic-Tech 서비스 개발 경험.
