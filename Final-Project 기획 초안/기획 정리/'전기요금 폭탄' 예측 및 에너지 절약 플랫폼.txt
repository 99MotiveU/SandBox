'전기요금 폭탄' 예측 및 에너지 절약 플랫폼

한 줄 요약: 실시간 사용량 기반으로 전기요금을 예측하고, 절약 방법을 시뮬레이션해주는 생활밀착형 플랫폼.
기획 배경: 매년 8~9월 반복되는 사회적 갈등 '전기요금 누진제' 문제를 데이터로 해결.
PoC 목표: 한국전력의 복잡한 누진제 요금 계산 로직을 완벽히 구현하고, 사용자의 데이터 입력을 통해 요금 예측 및 시뮬레이션 기능 제공.
핵심 기능:
현재 계량기 값 입력 시, 실시간 예상 요금 및 누진 구간 표시
"에어컨 2시간 더 사용 시" 등 시나리오 기반 요금 변동 시뮬레이션
우리 아파트 단지 내 사용자 간 익명 사용량 비교 랭킹
마이크로서비스 분리 방안:
서비스 A (사용자/인증): 사용자 가입/로그인, 가구 정보(주소, 계약종별) 관리.
서비스 B (요금/계산): 핵심 로직. 전기 사용량을 받아 요금을 계산하는 복잡한 정책 로직 전담.
서비스 C (통계/커뮤니티): 사용자별/단지별 통계 데이터 생성, 절약 팁 공유 커뮤니티 기능.
기대 학습 효과: 복잡한 현실 세계의 정책(규칙)을 코드로 구현하는 능력, 데이터 시각화 API 설계.
